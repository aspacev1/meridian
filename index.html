Collateral Asset Type collateral_code VARCHAR(10) Details ›
Definition
Standardised classification code identifying the type of asset pledged as collateral for a loan. Determines the collateral recognition percentage and haircut applied in Basel IV RWA calculation. Mapped from CBS collateral classification reference table.
Calculation
